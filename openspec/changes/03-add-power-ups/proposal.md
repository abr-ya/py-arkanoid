## Why

Power-ups add moment-to-moment variety after the base game and level system are stable. Keeping them separate protects the first playable slice from timer, multi-ball, and stacking complexity.

## What Changes

- Add falling bonus items from eligible bricks.
- Add catch detection between paddle and bonus items.
- Add Wide, Slow, Multi, and Sticky effects.
- Add timers and expiration behavior for timed effects.

## Capabilities

### New Capabilities

- `power-ups`: Falling bonus items, activation, stacking rules, and effect expiration.

### Modified Capabilities

None.

## Impact

- Extends core session state with active effects and optional multiple balls.
- Extends rendering with bonus item visuals and active-effect indicators.
